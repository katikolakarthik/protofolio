Attendance Management System

Overview: This is a full-stack web application developed to manage and track student attendance efficiently. It simplifies the attendance-taking process for faculty and provides real-time reports and analytics.

Key Features

  • Role-based login system for Admin, Faculty, and Students
  • Faculty can mark and update attendance daily
  • Students can view their attendance status
  • Admin can manage users, classes, and attendance data
  • Attendance reports generated monthly/semester-wise
  • Responsive UI for mobile and desktop users

Tech Stack:

  • Frontend: React.js, HTML, CSS
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: JWT (JSON Web Tokens)

Learning Outcome

Through this project, I gained hands-on experience in full-stack development using the MERN stack, RESTful API integration, role-based authentication, and CRUD operations. It helped me understand real-world application structures and deployment practices.
